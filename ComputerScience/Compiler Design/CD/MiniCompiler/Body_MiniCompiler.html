Mini Compiler


This minicompiler works with the differnt phasses of scanning (cic.l), parsing (cic.y) and intermediate code generation (till this it has been covered in appendix-A).The file codegen.c has the necessary routines to generate the machine code from the intermediate code. The machine code generation for assignement statements and add experesion is presented.The machine code generation for the remaining intermediate code can be developed in a similar manner.
