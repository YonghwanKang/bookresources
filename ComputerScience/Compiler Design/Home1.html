The CD contains the following contents:

  • Source code for generating intermediate code for C as discussed in Chapter 6.
  • Source code for generating 8086 machine code as discussed in Chapter 8.
  • Source code for generating intermediate code for C using Lex and Yacc tools for scanning and parsing as discussed in Appendix-A.
  • Source code for generating intermediate code for C using JavaCC for scanning and parsing as discussed in Appendix-B.
  • A section on Laboratory Exercises along with their source codes.
  • Source code for a very simple mini compiler for C language.

(Only few features of the languages are covered; the rest is left to the reader to practice.)

Note: All these codes are tested in both Linux (using CentOS and Fedora) and Windows platforms (using Cygwin).

HOW TO RUN THE CODES
The steps for running the codes in both Linux and Windows are as follows:
In Linux:

  • Go to the folder where the files (source code, test input C files) are saved.
  • Run the make command

(Before executing the make command, open the file makefile and study the linkage of all the files involved.)
In Windows:
Install the Cygwin software (freely downloadable from www.cygwin.com)
The home folder for the Cygwin may look as follows:
D:\cygwin\home\muni\compiler\sourcecode

  • Go to the Cygwin home folder in the command line
  • Create any sub folder based on your requirements
  • Copy all the required source files into this folder
  • Run the make command
  • Run the exe file
