Advanced 3D Game Programming Using DirectX 8.0

by Peter Walsh, Adrian Perez and Dan Royer

 

Thank you for purchasing this book.  We tried to pack this CD full of goodies that we hope you find it useful.

 

The CD material is divided into the following directories:

 

Applications

  • Paint Shop Pro Anniversary Edition (30-day evaluation)

 

Book Code

In the Book Code directory you'll find the source code for Chapters 1-9, plus the game "Mobots Attack!".  Each chapter (and the game) has its own workspace so you can use them independently of each other. The Book Code directory has information to help you compile the projects.

 

DirectX 8.0a SDK

In order to play with any of the book code, you'll need the DirectX SDK installed.  The 8.0a version is provided on this CD.

 

Documents

Several people have contributed additional material for you to read. The following articles are contained in this directory.

  • Improving Shadows and Reflections via the Stencil Buffer (By Mark J. Kilgrad, courtesy of http://www.nvidia.com)
  • Multithreading 1 &2  (By Dan Royer, courtesy of http://www.flipcode.com)
  • Harmless Algorithms Issues 1-3 (By Edward Kmett, courtesy of http://www.flipcode.com)
  • Radiosity in English 1&2 (By Paul Nettle, courtesy of http://www.flipcode.com)
  • Building a 3D Portal Engine (By Jacco Bikker, courtesy of http://www.flipcode.com)
  • Defeating Lag with Cubic Splines (By Nick Caldwell, courtesy of http://www.gamedev.net)
  • Understanding C++ Exception Handling (By Steve Crocker, courtesy of http://www.gamedev.net)
  • Using Interfaces with DLLs (By Gaz Iqbal, courtesy of http://www.gamedev.net)
  • Why Pluggable Factories Rock My Multiplayer World (By Mason McCuskey, courtesy of http://www.gamedev.net)
  • Quake 2 BSP File Format (By Max McGuire)
  • Generic Collision Detection for Games Using Ellipsoids (By Paul Nettle)

 

 

Other Code

This directory contains other source code projects that you may find useful.

  • Crystal Space (An open-source, multi-platform 3d engine)
  • Descent 1 & 2 (The full source - Thanks parallax) 
  • Wolfenstein 3D (One of the first of id's masterpieces)
  • Doom (Does it even need a description?  The classic from id)
  • Quake I (The GPL'ed source code to id software's masterpiece)
  • Quake III (Again, more source code from id, this time version 3)
  • Stan Melax (ProgMesh) (Stan Melax's progressive mesh code that appeared in Game Developer magazine)
  • STLport (An exceedingly nice implementation of the STL, strongly recommended if you want to play with "Mobots Attack!")
