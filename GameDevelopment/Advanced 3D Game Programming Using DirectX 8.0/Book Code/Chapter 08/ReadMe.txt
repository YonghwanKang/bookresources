The chapter 8 workspace has the following projects
MultiTex (Application)
Detail (Application)
GameLib (Library)
Math3D (Library)

/******************************************************************/
Application Name: Multitex

Application Description: 
    Shows 6-pass multitexture in action.  Only one pass is on by 
    default, the rest can be turned on.  Currently does not work
    on the voodoo1, voodoo2, or voodoo banshee.



/******************************************************************/
Application Name: Detail

Application Description: 
    Shows detail texturing in action.  Currently does not work
    on the voodoo1, voodoo2, or voodoo banshee.
