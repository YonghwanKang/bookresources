README.TXT for SOURCE files

All of the source code for this book has been written and tested under
Blitz Basic. You can get this program from the PROGRAMS directory on the 
CD.

I. INSTALLING THE SOURCE

To install the source, simply drag the entire directory into the 
desired place on your hard drive and make a copy. Make sure to clear the
Read-Only flag by right clicking on the directory and deselecting the
Read-Only checkbox.
