This demo requires  NV30 class hardware, e.g. Nvidia Quadro Fx 1000/2000 or Geforce FX card. 
It utilises OpenGL and the NV30 Cg profile.

To build this project you will require the following files, which are part of the Nvidia OpenGL SDK:

data_path.cpp
MovieMaker.cpp
nv_dds.cpp

MovieMaker.h


The Nvidia OpenGL SDK is available for free download from http://www.nvidia.com/developer
